# BoatSpotMedia Dashboard Requirements

## Creator Dashboard

### Navigation Layout
- Overview
- Sales
- Payouts
- My Videos
- Upload Batch
- Analytics
- Tax Center
- Settings

### Overview
Must show at a glance:
- Total earned this month
- Total earned lifetime
- Videos sold this month
- Videos sold total
- Pending payouts
- Last sale
- Platform commission applied
- Earnings Overview: Today / This week / This month / Lifetime

### Sales Analytics
Filters:
- Today
- This week
- This month
- Custom date range

Show:
- Sales by day
- Sales by video
- Gross revenue
- Platform commission
- Stripe fees
- Net earnings
- Taxes withheld, if applicable

Buttons:
- Export CSV
- Export PDF

### Payouts
Show:
- Sent payouts
- Pending payouts
- Estimated transfer date
- Payment method: Stripe
- Stripe status: Connected / Not connected
- Last payout date
- Next payout estimate

### Video Performance
Show per video:
- Preview views
- Clicks
- Conversion rate
- Total sales
- Revenue by video

### Earnings Breakdown
Show clearly:
- Gross revenue
- Platform commission
- Stripe fees
- Net earnings

### Notifications Panel
Example notification:
- Your video sold
- Original / file name / location
- $40 earned

### Account Settings
Allow creator to edit:
- Name
- Email
- Password
- Stripe connection
- Language
- Currency display

### Demand Insights
Future section to show:
- Search demand by location
- Most searched dates
- Locations with buyer interest but low creator supply

## Owner Dashboard

### Overview
Must show:
- Sales today
- Sales this week
- Sales this month
- Commission earned today / month / lifetime
- New creators registered
- New buyers registered
- Videos uploaded today
- Videos sold today
- Active alerts

### Creators Manager
Owner can:
- View creators list
- Edit name / email
- Reset password
- Activate / deactivate account
- Temporarily suspend
- Delete account
- View number of videos uploaded
- View total sold
- View total generated
- View Stripe status: connected / incomplete
- View last activity
- View creator filming locations
- View recent batches

### Buyers Manager
Owner can:
- View purchase history by buyer
- Edit buyer email
- Resend download link
- Regenerate video access
- Delete account upon request
- View recent purchases

### Sales Control Panel
Show:
- Individual sale
- Creator
- Buyer
- Amount
- Platform commission
- Creator net
- Date
- Payment status

### Video Control Panel
Owner can:
- View all videos
- Search by creator
- Search by date
- Search by location
- Delete video
- Hide video without deleting it
- Edit metadata
- Change preview if it fails
- Disable video visibility

### Internal Support Center
Show alerts for:
- Download links not delivered
- Failed payments
- Incomplete Stripe accounts
- Corrupted videos
- Failed uploads

### Stripe Control
Show:
- Creator connected correctly
- Pending payout
- Failed payout
- Restricted account

### Global Analytics
Show:
- Sales by location
- Top searched locations
- Top creators leaderboard
- Top seller this week example: MarineFootagePro – $420

### Commission Control
Owner can:
- Change global platform commission
- Change commission by creator
- Create temporary promo commission

### Admin Permissions
Roles:
- Owner
- Admin
- Support
- Finance

Only Owner has full access.

### Critical Buttons
- Suspend creator
- Delete video
- Resend download link
- Issue refund
- Edit buyer email
- Delete incomplete batch
- View payout status
- Disable video visibility
